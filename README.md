[ENG]
An iOS application developed with SwiftUI that leverages Artificial Intelligence to reimagine interior spaces. Users can upload a photo of their room, select from various architectural styles (Modern, Cyberpunk, Minimalist, etc.), and generate a redesigned version of their space.

Key Features:

Built with SwiftUI and MVVM architecture.

Modern and responsive UI with glassmorphism effects.

Integrated PhotosUI for seamless image selection.

Async/Await for handling asynchronous tasks.

Dark Mode support and dynamic theme adaptation.

In-app gallery to save and view past designs.

Tech Stack: Swift, SwiftUI, MVVM, Core Graphics, PhotosUI.

[TR]
SwiftUI kullanılarak geliştirilmiş, iç mekanları yapay zeka ile yeniden tasarlayan bir iOS uygulaması. Kullanıcılar odalarının fotoğrafını yükleyip farklı mimari tarzlardan (Modern, Cyberpunk, Minimalist vb.) birini seçerek mekanın yeniden tasarlanmış halini oluşturabilirler.

Öne Çıkan Özellikler:

SwiftUI ve MVVM mimarisi ile geliştirildi.

Glassmorphism efektleri içeren modern kullanıcı arayüzü.

Görsel seçimi için PhotosUI entegrasyonu.

Asenkron işlemler için Async/Await yapısı.

Karanlık Mod (Dark Mode) desteği.

Geçmiş tasarımların kaydedildiği galeri sistemi.

Teknolojiler: Swift, SwiftUI, MVVM, Core Graphics, PhotosUI.
